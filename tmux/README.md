## tmux keybindings

### windows

| keybind | operation                      |
| ------- | ------------------------------ |
| w       | ウィンドウの一覧選択           |
| '       | 入力番号のウィンドウへ移動     |
| l p     | 1 つ後のウィンドウへ移動       |
| n       | 1 つ前のウィンドウへ移動       |
| f       | ウィンドウの検索               |
| j       | ウィンドウの並び替え(1 つ左へ) |
| l       | ウィンドウの並び替え(1 つ右へ) |

### pane

| keybind  | operation                    |
| -------- | ---------------------------- |
| h        | 左右にペイン分割             |
| q        | ペイン番号を表示             |
| !        | ペインを解除してウィンドウ化 |
| o        | ペインを順に移動             |
| ;        | 以前のペインへ移動           |
| space    | レイアウトを変更             |
| {        | ペインの入れ替え（前方向）   |
| }        | ペインの入れ替え（後方向）   |
| ctrl + o | ペインの入れ替え（全体）     |
| t        | ペインに時計を表示           |
